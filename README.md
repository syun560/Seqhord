## Seqhord
- Seqhordはブラウザで動作するサウンドプログラミング環境です。
- WEB MIDI APIを経由して音を再生することができます。
- VOICEVOXと連携し歌声を生成します。

## できること - Features
- 記述された譜面からSMF（標準MIDIファイル）、MusicXML（メロディのみ）等のファイルを生成します。
