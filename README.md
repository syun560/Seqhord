## Seqhord
- Seqhordは自作のサウンドプログラミング環境です。
- 音を再生することができます。
- VOICEVOXと連携しプレビューとして歌声を生成します。

## できること - Features
- 記述された譜面からSMF（標準MIDIファイル）、MusicXML（NEUTRINO用）等のファイルを生成します。
- creates MIDI, MusicSML(for NEUTRINO) file
